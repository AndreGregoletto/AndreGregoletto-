Método usado

A página usa Jackson-Pollock 3 dobras quando os pontos medidos batem com o protocolo. Depois converte densidade corporal para BF% pela equação de Siri.

Homens — 3 dobras

S = peito + abdômen + coxa
BD = 1.10938 - 0.0008267*S + 0.0000016*S² - 0.0002574*idade
BF% = 495 / BD - 450

Mulheres — 3 dobras

S = tríceps + supra-ilíaca/flanco + coxa
BD = 1.0994921 - 0.0009929*S + 0.0000023*S² - 0.0001392*idade
BF% = 495 / BD - 450

Fórmula de adipômetro é estimativa. Para menores de 18 anos ou idosos fora da faixa original, o resultado é exibido como referência visual, não avaliação clínica.
